Задачи на работу с файловой системой
1. Написать функцию ХХХ (имя подставьте своё), которая на вход принимает
необязательный параметр - число, по-умолчанию равное 0. Значение параметра
необходимо приводить к числу.
В функции необходимо создать файл (в последующем обращении - открыть
существующий) и записывать значение параметра в файл.
Если файл пустой , то запишется число из параметра.
Если файл не пустой , то к значению в файле необходимо прибавить значение в
параметре и записать в тот же файл. Например, в файле хранится число 5, в
параметре 2, в файл запишется 7.
2. Даны два файла ХХХ и YYY. В файлах записаны тестовые слова (на латинице,
придумайте сами). разделенными пробелами. Необходимо написать функцию,
которая создаст новый три файла:
only_first.txt : содержащий строки, которые встречаются только в первом файле;
both.txt : только строки, которые встречаются в обоих файлах;
more_two.txt : только строки, которые встречаются в каждом файле более двух раз.
3. Дан файл со словами. Упорядочить слова по алфавиту и записать в тот же файл.
4. Дан файл. Каждая строка содержит имя, пароль и email, разделенные символами ':'
(двоеточие).
Например:
john:12345: john@gmail.com
kate:12345: kate@gmail.com
test:21345: test@gmail.com
mike:12145: test@gmail.com
Удалить строки с повторами email и строки, в которых совпадают имена (т.е. удалятся
две последних записи).
5. Написать функцию, которая будет показывать список всех файлов в данной папке в
виде дерева, как показано на ниже (поиск файлов происходит и во всех вложенных
уровнях).
root dir
-- dir 1
-- dir 2
---- dir 2.1
---- dir 2.2
------ dir 2.2.1
---- dir 2.3
6. ( +2 ) На вашем сайте необходимо сделать авторизацию пользователя. Для этого
необходимо в верхней части сайта (header) добавить две ссылки: Вход и Регистрация.
http://joxi.ru/krDldd5FE4Jl9r
Ссылки ведут на файл sign_in.php (вход на сайт) и sign_up.php (регистрация).
На обеих страницах необходимо отобразить форму с тремя полями: username, email и
password.
При переходе на страницу Sign Up после ввода пользователем данных эти данные
необходимо записать в файл users.txt в виде:
username:email:password
где:
- username , имя пользователя (только на латинице)
- email , является уникальным
- password , пароль пользователя
Перед записью необходимо проверить, есть ли такой пользователь в файле.
Подсказка : создайте три функции для работы с данными из файла.
Первая функция (например, getUsers()) будет считывать данные из файла и
возвращать их в виде массива.
Вторая функция (например, checkUser($email, $users), где $email - это email
пользователя, полученный из формы регистрации, а $users - результат функции
getUsers(), т.е. список пользователей) будет сверять введенный пользователем при
регистрации email ($email) со списком пользователей ($users) и возвращать true если
пользователь с таким email уже есть, и false если такого не нашлось.
Третья функция (setUser($email, $password)) будет записывать нового пользователя в
файл.
При успешной регистрации (пользователь не нашелся и записался в файл) создать в
сессии переменную user и занести в нее данные пользователя: username, email.
После занесения данных пользователя в сессию сделайте редирект (с помощью
функции header) на главную страницу сайта.
Если во время регистрации пользователь уже был найден в файле, то выдайте
сообщение “Пользователь с таким email уже существует!”.
При переходе на страницу Sign In после ввода пользователем данных необходимо
проверить, есть ли такой пользователь в файле users.txt (воспользуйтесь
подготовленными функциями). Если пользователя нет, то выдайте сообщение
“Пользователя с таким email нет!”.
Если пользователь найден, то занесите его данные (username, email) в сессию и
сделайте редирект на главную страницу сайта.
При попадании на главную страницу сайта необходимо проверить сессию
пользователя. Если пользователь есть в сессии, то вместо ссылок Sign In и Sign Up
вывести:
Привет %username%!
7. ( +1 ) Создайте на сайте пункт меню “Users”. При клике на этот пункт в центральной
части выведите список зарегистрированных пользователей в виде:
Username: % username из файла %, email: % email из файла %
